What is HTML?

HyperText (Text that contains links) Markup (annotate) Language a way of marking up files so that browsers know how to display your content as a webpage. Hyper media is an extension of hypertext

HTML - structure CSS - style JS - behaviour or functionality

History of HTML

Before 1997, no standards were there. In 1997, World wide web consortium (w3c) started HTML 4. 2000 - XHTML 1 2004 (WHATWG - Web Hypertext Application Technology Group: HTML) 2007 - both started working together 2011 - HTML5

NOTE: The years are not exact

Anatomy of HTML

Opening Tag, Closing Tag, Content, Element br - line break, hr - horizantal rule Attributes id="" (unique) only in opening tag

  • class - applies special properties to groups of elements
  • unique id to just a single element on the page
  • Style lets you apply visual style.
  • an accesskey is very important for accessibility. It allows a shortcut key to kind of activate, or put into focus, an element
  • tabindex - order elements will come into focus using the tab key. A "high" tab ndex is actually a small number. So tabindex = "1" is the highest priority.
